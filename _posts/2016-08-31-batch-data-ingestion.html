In the batch data ingestion pattern, data is written to batch storage system such as Hadoop Data Lake. Once the data is written to the Data lake in batches, aggregation is done and the aggregated views are persisted and then written to a relational database designed for reporting (such as Data Warehouse) to let reporting done on the aggregated data.  Analysis can be done on the aggregated data and/or raw data from the Data Lake using Hive structures.
